# Privacy Policy for Import Manager

**Last updated:** April 21, 2026

Import Manager is a personal desktop application that helps you backup and manage files using Google Drive.

## Information We Access
- When you sign in with Google, the app requests access to your Google Drive files **only** to:
  - Read files and folders you choose
  - Upload/download backups
- The app **does not** collect any personal information.
- The app **does not** store your data on any external servers.
- All operations happen locally on your computer.

## Google User Data
We only use the Google Drive API to access files you explicitly select for backup or import.  
We do **not** share, sell, or use your Google data for any other purpose.

## Data Storage
- Your Google credentials are handled securely by Google's official OAuth system.
- No credentials or sensitive data are stored by Import Manager except for normal local configuration.
